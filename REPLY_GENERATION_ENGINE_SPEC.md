# REPLY_GENERATION_ENGINE_SPEC.md

## 1. 목적

이 문서는 대화코칭어플이 답장을 어떻게 생성하고 선별해야 하는지 정의한다.

앞으로 이 앱은 답장을 바로 3개 만드는 방식이 아니라, 여러 후보를 만들고 어색한 문장을 제거한 뒤 최종 3개를 보여주는 방식으로 발전해야 한다.

## 2. 기본 철학

답장은 정답이 아니다.
답장은 반응이다.

사람은 관계 상황에서 완성된 입장문을 보내지 않는다.
대부분 짧게 반응하고, 필요한 만큼만 감정을 드러낸다.

따라서 이 앱은 “성숙한 답장”보다 “실제로 보낼 수 있는 답장”을 우선한다.

## 3. 전체 처리 흐름

v0.5 이후 답장 생성 흐름은 아래와 같다.

1. 사용자 입력 수집
2. 상황 유형 분류
3. 말투 온도 선택
4. 후보 문장 여러 개 생성
5. 어색한 문장 제거
6. 너무 성숙한 문장 제거
7. 상담문 제거
8. 최종 3개 선택
9. 사용자에게 표시

## 4. 상황 유형

초기 상황 유형은 아래 6개다.

1. 호감 탐색 / 부담 없이 이어가기
2. 호감 탐색 / 조금 더 다정하게 표현하기
3. 텐션 하락 / 잘 모르겠다
4. 서운함·갈등 / 차분히 말하기
5. 선 긋기
6. 관계 애매함

## 5. 말투 온도

답장 후보는 아래 말투 온도 중 하나를 가진다.

### 가볍게 받기

상황을 키우지 않고 짧게 받는다.

예:
아 그랬구나

### 살짝 다정하게

호감이나 관심을 약하게 드러낸다.

예:
난 그냥 좀 궁금했어

### 담담하게 선 긋기

불편함이나 경계를 짧게 드러낸다.

예:
지금은 좀 어려워

### 흐름 보기

당장 결론 내리지 않고 대화를 가볍게 유지한다.

예:
오키 알겠어

### 살짝 찌르기

너무 공격적이지 않게, 사용자의 헷갈림을 살짝 드러낸다.

예:
너무 애매하면 나도 좀 헷갈리긴 해

## 6. 후보 생성 원칙

후보는 처음부터 3개만 만들지 않는다.

각 상황별로 최소 8개 이상의 후보를 만든 뒤, 아래 기준으로 걸러낸다.

걸러낼 문장:

- 너무 길다.
- 너무 성숙하다.
- 너무 상담문 같다.
- 너무 예의 바르다.
- 관계를 정리하려고 한다.
- 감정을 과하게 설명한다.
- 사용자의 감정을 낮춘다.
- 상대를 조종한다.
- 질투를 유발한다.
- 답장을 일부러 늦추라고 한다.
- 실제 카톡에서 안 쓸 것 같다.

## 7. 최종 선택 기준

최종 출력은 아래 3개다.

1. 가장 무난한 답장
2. 조금 더 다정한 답장
3. 선을 지키는 답장

각 답장은 서로 분명히 달라야 한다.

하지만 세 답장 모두 실제 사람이 보낼 수 있어야 한다.

## 8. 코칭 문구 생성 원칙

코칭 문구는 짧아야 한다.

좋은 코칭 문구:

- 지금은 바로 따지기보다 가볍게 받는 쪽이 나아요.
- 답장 하나로 관계를 확인받으려 하면 말이 무거워질 수 있어요.
- 서운함은 말해도 되지만, 첫 문장부터 세게 던질 필요는 없어요.
- 거절은 길게 설명할수록 다시 맞춰주는 흐름이 될 수 있어요.

나쁜 코칭 문구:

- 상대가 방어적으로 느낄 수 있습니다.
- 관계 리스크를 낮추는 방향입니다.
- 감정 과잉을 방지합니다.
- 사용자 의도를 부드럽게 반영했습니다.
- 이 답장이 적합합니다.

## 9. v0.5 구현 방향

v0.5에서는 app.js에서 아래 구조를 구현한다.

1. 상황 유형을 판별한다.
2. 해당 상황의 후보 문장 풀을 불러온다.
3. 금지 표현이 들어간 후보를 제거한다.
4. 너무 긴 후보를 제거한다.
5. 가장 무난한 답장, 조금 더 다정한 답장, 선을 지키는 답장을 각각 선택한다.
6. 답장 문장과 짧은 코칭 설명을 화면에 보여준다.

## 10. 최종 목표

대화코칭어플의 답장은 아래 질문을 통과해야 한다.

“이 문장을 실제 카톡에 붙여넣을 수 있는가?”

이 질문을 통과하지 못하면 아무리 안전하고 성숙한 문장이어도 실패다.
