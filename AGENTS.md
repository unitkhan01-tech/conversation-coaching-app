# AGENTS.md

## 기본 전제

사용자는 코딩을 전혀 모르는 비개발자다.
사용자는 Run 버튼, 터미널 명령, 설치 명령, 실행 명령의 의미를 구분하지 못한다.
따라서 초기 단계에서는 절대로 터미널 명령을 실행하거나 제안하지 않는다.

## 절대 금지

- npm install 실행 금지
- npm run 실행 금지
- npx 실행 금지
- node 실행 금지
- python 실행 금지
- git 실행 금지
- pnpm 실행 금지
- yarn 실행 금지
- brew 실행 금지
- curl 실행 금지
- Run 버튼이 생길 수 있는 행동 금지
- 사용자의 명시 허가 없는 자동 실행 금지

## 작업 방식

1. 먼저 문서와 설계를 만든다.
2. 그다음 파일 구조를 제안한다.
3. 그다음 코드를 생성한다.
4. 실행이 필요한 경우에는 먼저 이유, 위험, 사용자가 눌러야 하는 버튼을 설명하고 멈춘다.
5. 사용자가 명시적으로 허락하기 전까지 실행하지 않는다.

## 제품 철학

이 앱은 상대를 조종하거나 밀당 기술을 가르치는 앱이 아니다.
이 앱은 사용자가 감정적으로 과잉 반응하거나, 너무 차갑게 굴거나, 불필요하게 관계를 망치는 메시지를 보내기 전에 한 번 더 생각하게 만드는 대화 코칭 도구다.

좋은 답장은 상대를 이기기 위한 말이 아니다.
좋은 답장은 관계의 맥락과 사용자의 감정을 모두 덜 해치도록 조율된 말이다.

## 품질 기준

항상 아래 기준으로 결과를 점검한다.

- 부담도
- 자연스러움
- 실제 사용 의향
- 관계 리스크
- 감정 과잉 여부
- 상대방 통제/조종 느낌 여부
- 사용자가 후회할 가능성
